## 折射 refraction

1. 折射率 $n$ $(refraction\ index)$
	- $formular$
		1. $n_1\cdot\sin{\theta_1}=n_2\cdot\sin{\theta_2}$
		2. $n_1\cdot v_1 = n_2\cdot v_2$
2. 光学密度 $(optical\ density)$
3. $total\ internal\ reflection$ (全反射)
	- $i=\theta_c$ $(critical\ angle)$

	1. $n_{大}\to n_{小}$
	2. $i>\theta_c$
	3. $\theta_c = ?$

| substance | n    |
| --------- | ---- |
| vacuum    | 1    |
| air       | 1    |
| water     | 1.33 |
| glass     | 1.5  |
